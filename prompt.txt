  Python 代碼摘要：Health Checkup System (Powerpuff Girls Edition)

  這是一個基於 Tkinter 開發的桌面應用程式，專為自動化處理健檢專案、動態加選管理及 PDF
  報告生成而設計。其核心邏輯與架構如下：

   1. 資料驅動架構：
       * 資料源：直接讀取 中心全組套.xlsx，支援分頁讀取 (主方案與加選清單)。
       * 動態解析：程式會自動解析 Excel 中的列 (方案) 與行 (項目)，支援多種性別特定方案。

   2. GUI 互動邏輯 (Powerpuff Girls Theme)：
       * 主題配色：採用「飛天小女警」風格的粉色、藍色、綠色 UI 方案。
       * 動態列表：使用者可透過 Combobox 切換方案，主表格會即時連動更新項目與價格。
       * 加選機制：右側面板提供獨立的勾選清單，支援動態增加加選項目至總清單中。
       * 退選邏輯：雙擊主表中的項目可執行退選並自動扣款，維持金額精確度。

   3. 專業報告生成 (PDF Engine)：

       * 中文字型集：透過 msyh.ttf (微軟雅黑) 支援繁體中文渲染。
       * 自動化排版：具備類別排序 (CATEGORY_ORDER)、表格樣式設計 (TableStyle) 及分頁溢出處理。
